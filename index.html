HIDDEN

Filter websites to automatically open them in Incognito mode

Konuko II - Website | Github

Instructions

  • Simply add keywords or URLs that you would like to be always opened by Incognito window.
  • For example, if you add the keyword "hacker" to your filters, every website containing the word "hacker" in it will be immediatly opened under incognito mode. (eg. "https://en.wikipedia.org/wiki/Hacker" or "https://www.hackerrank.com/")
